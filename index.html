Installation

To install Crimson, run the following command:

curl -sSL https://crimson.sh/install.sh | bash

Documentation

Crimson provides a simple CLI interface:

  • crimson init - Initialize a new project
  • crimson install <package> - Install a package
  • crimson publish - Publish your package
  • crimson config - Configure settings
